# Student-Course Management System

## 📌 Project Idea

A Laravel-based system to manage students, courses, and their relationships with each other.

### Core Features

- **Student Management**
  - Add new students
  - View student lists
  - Edit student records
  - Delete students

- **Course Management**
  - Create courses
  - Browse all courses
  - Update course details
  - Remove courses

- **Relationship Management**
  - Enroll students in courses
  - View enrollments
  - Manage student-course links
  - Track which students are in which courses
